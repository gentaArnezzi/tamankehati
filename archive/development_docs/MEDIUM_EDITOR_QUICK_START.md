# Quick Start: Medium-Style Editor ✨

## 🎯 Sudah Siap Pakai!

Editor artikel style Medium.com sudah terintegrasi di dashboard Anda!

---

## 📍 **Cara Mengakses**

1. Login ke dashboard
2. Menu: **Taman** → **Berita/Artikel** → **Buat Artikel**
3. URL: `/dashboard/taman/berita/create`

---

## ✏️ **Cara Menulis**

### **1. Cover Image** (Optional)
- Klik area dashed di atas
- Masukkan URL gambar
- Atau skip untuk artikel tanpa cover

### **2. Judul** (Required)
- Large input di bagian atas
- Font besar seperti Medium
- Maksimal 60-80 karakter

### **3. Excerpt** (Optional)
- Subtitle di bawah judul
- Ringkasan 1-2 kalimat
- Akan muncul di list artikel

### **4. Content** (Required)
- Area besar untuk menulis
- Gunakan toolbar untuk formatting
- Atau tulis dengan markdown

---

## 🎨 **Toolbar Formatting**

### Quick Buttons:
| Button | Fungsi | Markdown |
|--------|--------|----------|
| **B** | Bold | `**text**` |
| **I** | Italic | `*text*` |
| **T** | Heading | `## text` |
| **•** | List | `- text` |
| **"** | Quote | `> text` |
| **<>** | Code | `` `text` `` |
| **📷** | Image | `![](url)` |

---

## 💾 **Save & Publish**

### Top Bar (kanan atas):

1. **⋯ (More)** → Settings sidebar
   - Set kategori
   - Edit metadata
   
2. **Save** → Simpan sebagai draft
   - Artikel tersimpan
   - Bisa diedit lagi nanti
   
3. **Publish** → Publikasikan
   - Langsung live
   - Muncul di website

---

## 📝 **Contoh Penulisan**

```markdown
## Introduction
Taman Nasional merupakan...

## Keunikan Flora
Beberapa spesies endemik:
- **Rafflesia arnoldii**
- *Nepenthes rajah*
- Kantong semar raksasa

## Data Penelitian
> "Biodiversitas di kawasan ini mencapai 
> 2,500 spesies." - Dr. Ahmad

## Kesimpulan
Konservasi sangat penting untuk...

![Pemandangan](https://example.com/image.jpg)
```

---

## 🎯 **Tips**

### ✅ DO:
- Gunakan cover image yang menarik
- Buat judul yang engaging
- Break content dengan headings
- Add images untuk visual break
- Keep paragraphs short (2-3 kalimat)

### ❌ DON'T:
- Jangan tulis paragraf panjang
- Jangan lupa save berkala
- Jangan skip excerpt (penting untuk SEO)

---

## 🚀 **Workflow Cepat**

1. **Draft Mode** (First)
   ```
   1. Tulis judul
   2. Tulis outline
   3. Save as draft
   ```

2. **Content** (Second)
   ```
   1. Tulis intro
   2. Expand points
   3. Add images
   4. Save draft
   ```

3. **Polish** (Third)
   ```
   1. Review & edit
   2. Add cover image
   3. Write excerpt
   4. Set category
   5. PUBLISH!
   ```

## 🎨 **Style Guide**

### **Typography**
- Judul: 42px, Bold
- Content: 20px, Georgia
- Line height: 1.8

### **Best Practices**
1. **Judul**: 6-10 kata
2. **Excerpt**: 150-200 karakter
3. **Paragraf**: 2-3 kalimat
4. **Images**: 1 cover + 2-3 inline
