# 📁 Archive Directory

Dokumentasi historis dan referensi yang telah diorganisir berdasarkan kategori.

## 📂 Struktur Folder

### `/deployment/`
Dokumentasi deployment untuk berbagai platform:
- `DEPLOYMENT_GUIDE.md` - Panduan deployment umum
- `DEPLOYMENT_QUICKSTART.md` - Quick start deployment
- `DEPLOYMENT_README.md` - Dokumentasi deployment
- `DEPLOYMENT_SETUP.md` - Setup deployment
- `DEPLOYMENT_SUMMARY.md` - Ringkasan deployment
- `RENDER_DEPLOY.md` - Deployment ke Render.com
- `VERCEL_DEPLOY.md` - Deployment ke Vercel
- `RAILWAY_DEPLOYMENT.md` - Deployment ke Railway
- `DEPLOY_RECOMMENDATION.md` - Rekomendasi deployment

### `/features/`
Dokumentasi implementasi fitur:
- **Park Management**: `PARK_*.md`
- **Gallery**: `GALLERY_*.md`
- **Dashboard**: `DASHBOARD_*.md`
- **Tour/Onboarding**: `TOUR_*.md`, `INTERACTIVE_*.md`, `PRODUCT_*.md`
- **Flora/Fauna**: `FLORA_*.md`
- **Approval System**: `APPROVAL_*.md`, `CASCADE_*.md`
- **Maps**: `MAP_*.md`, `NOMINATIM_*.md`, `LEAFLET_*.md`
- **UI/UX**: `MINIMAL_*.md`, `MODERN_*.md`, `TAMANKEHATI_*.md`

### `/fixes/`
Dokumentasi perbaikan bug dan issue:
- Semua file dengan pattern `*_FIX.md`
- `*_FIX_FINAL.md` - Versi final dari perbaikan
- `SETTINGS_LOGOUT_BUG_FIX.md` - Perbaikan bug logout

### `/sop/`
Standard Operating Procedures:
- `SOP_LENGKAP_INPUT_DATA.md` - SOP input data lengkap
- `SOP_PENGELOLAAN_DATA_TAMAN_KEHATI_2025.md` - SOP pengelolaan data
- `SOP_REGIONAL_ADMIN_LENGKAP.md` - SOP untuk regional admin

### `/analysis/`
Dokumen analisis dan rekomendasi:
- `*_ANALYSIS.md` - Analisis sistem
- `CHART_ANALYTICS_RECOMMENDATIONS.md` - Rekomendasi analytics
- `DASHBOARD_ANALYSIS.md` - Analisis dashboard

### `/development_docs/`
Dokumentasi development yang sudah ada sebelumnya (tidak dipindahkan).

---

## 🔍 Cara Mencari Dokumentasi

### Berdasarkan Topik
- **Deployment**: Lihat `/deployment/`
- **Fitur Baru**: Lihat `/features/`
- **Bug Fix**: Lihat `/fixes/`
- **SOP**: Lihat `/sop/`
- **Analisis**: Lihat `/analysis/`

### Berdasarkan Waktu
File diurutkan berdasarkan nama (yang biasanya mengandung tanggal atau versi).

---

## 📝 Catatan

- Dokumentasi ini adalah **historical reference** dan mungkin sudah tidak relevan dengan implementasi saat ini
- Untuk dokumentasi aktif, lihat folder `docs/`
- Jika butuh referensi implementasi lama, cek folder sesuai kategori
